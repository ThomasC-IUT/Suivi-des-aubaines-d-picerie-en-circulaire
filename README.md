# Suivi-des-aubaines-d'épicerie-en-circulaire
Projet web du cours 8WEB101 (Conception et programmation de sites Web)
Technologies utilisées pour concevoir le site web: Supabase, Github, ChartJS, le tout en Vanilla JS

Pour tester le site, il faut se rendre sur le lien Github. https://thomasc-iut.github.io/Suivi-des-aubaines-d-picerie-en-circulaire/index.html

Fonctionnalités: 
-Filtrage par magasin, catégories, prix
-Regroupement des circulaires par semaine
-Barre de recherche
-Mode compact
-Panier
-Historique de prix (Graphique)
-Statistiques
-Redirection vers les sites marchands
-Export panier (PDF)
